# Basic Chatbot 🤖

A simple rule-based chatbot built with Python as part of the **CodeAlpha Python Programming Internship**.

## How to Run

```bash
python chatbot.py
```

> No external libraries needed. Works with Python 3.x out of the box.

## How It Works

- You type a message in the terminal
- The bot matches your input against a set of predefined triggers
- It replies with a relevant response
- Type `bye` or `goodbye` to exit

## What It Can Handle

| Topic | Example Input |
|---|---|
| Greetings | "hello", "hi", "hey" |
| How are you | "how are you", "how's it going" |
| Name | "what is your name", "who are you" |
| Jokes | "tell me a joke", "make me laugh" |
| Time | "what time is it", "current time" |
| Date | "today's date", "what day is it" |
| Python | "do you like python" |
| Help | "what can you do", "help" |
| Thanks | "thank you", "thanks" |
| Exit | "bye", "goodbye", "quit" |

## Sample Conversation

```
You: hello
CodeBot: Hey there! How can I help you?

You: tell me a joke
CodeBot: Why do programmers prefer dark mode? Because light attracts bugs!

You: bye
CodeBot: Goodbye! Have a great day!
```
